HTML

  • HTML is used to create the structure of a webpage. It is made up of elements which work together to provide content and the basic framework for all pages on the web.
  • An element is a type of content on a webpage-TEXT,IMAGE,HEADER,LIST
  • Elements are also used to divide the content on the page and to describe the content to browsers and other developers-META,CONTENT DIVISION,SEMANTIC.
  • The head element contains information about the webpage, metadata.
  • HTML Attributes defines additional info about an element.
  • The title element defines the title for the webpage, important for SEO.
  • The style element defines the CSS styles
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML doc.
  • The script element embeds client-side scripts in the HTML doc.
  • the meta elements-charset,http-equiv,viewport. The viewport is the user's visible area of the webpage.
  • The body element represents the visible content shown to the user.
  • SEO stands for Search Engine Optimization
  • This “parent/child” language is often used by developers to explain the relationship among elements in HTML.
  • Child elements are nested inside the parent element.
  • In semantic HTML elements, the name of the element describes its content and the role it plays on a webpage.
  • Semantic also plays an important role in making sure your webpage is accessible to the widest variety of users, including those who use assistive technologies such as screen readers.
  • Non-semantic elements-DIV,SPAN
  • An element refers to the entire element from opening tag to closing tag
  • A tag refers only to what appears inside the angle brackets.
  • Headings are larger or more prominent titles and subtitles to organize a page.
  • The p element element represents a paragraph or block of text. You'll use this element extensively to wrap most of the text on your webpages.
  • List elements can be ordered, unordered, and list.
  • The image element contains information about images that are displayed on the webpage, it can hold various attributes.
  • The source element attribute within the image, is the most important because it defines the location of the image,
  • The break element creates a line of empty space, or a line break, between two blocks of content.
  • The `alt` attribute contains a text string that describes the appearance and functionality of the image.
  • The footer element appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.
  • Section elements are semantic elements that define a section in a document that contains thematic content
  • Each section element has an id attribute that also matches one of the four topics. HTML uses the id attribute as a way to identify elements and select them.

CSS

  • CSS is a programming language used by devs to define how a webpage's content is presented to users.
  • CSS defines colors, aesthetics, visual layout, organization.
  • The 3 ways to style a webpage are 1) inline css, 2) internal CSS style sheet, 3) external CSS style sheet.
  • When using a external CSS, we link the file using link element placed in the HTML's head element.
  • CSS rule is made up of Selector + Declarations that have 2 components 1) CSS property + 2) Value of the property.
  • Selector defines the element or attributes to which the rules, or declarations will apply.
  • Every HTML element is a box, CSS Box Model.
  • DRY- Don't Repeat Yourself.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Block vs. Inline Box. Outer diplay and Inline Display.
  • A link element creates the connection to an external file. rel attribute specifies the relationship between the current document and the linked resource.
  • The href attribute specifies the location(URL) if the external resource using a relative path.
  • The Class selector allows us to share a defined CSS rule an apply it to any element we choose by assigning that rule to a class attribute with a class selector.
  • The class attribute can be assigned to the element in the HTML.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is case sensitive.
  • The return statement tells the browser to return the result variable out of the function so it is available to use
  • Real interactivity on a website requires event handlers. These are code structures that listen for activity.
  • JS is a dymanic and weak typed lang. Dynamic: any variable can be assigned and reassigned values of all types.
  • JS weak typed lang: allows implicit type conversion when an operation involves mismatched types instead of throwing type errors.
  • Console refers to a test environment that developers use to checkout their code.
  • JS is executed from top to bottom.
  • A value is falsy: 0, -0, on, "", null, undefined, NaN or Boolean with False set.
  • Default value is FALSY.
  • Array is a single variable that is used to hold a group of data.
  • In order to use for loop we need the index + arrays length.
  • Function a set of instructions that tells the computer how to perform a certain task.
